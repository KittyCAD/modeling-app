Convert a number to inches from its current units.

```kcl
units::toInches(@num: number(Length)): number(in)
```



### Arguments

| Name | Type | Description | Required |
|----------|------|-------------|----------|
| `num` | `number(Length)` | A number. | Yes |

### Returns

`number(in)` - A number.
