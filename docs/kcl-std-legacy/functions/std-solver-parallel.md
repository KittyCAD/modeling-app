Constrain lines to be parallel.

```kcl
solver::parallel(@input: [Segment; 2+])
```



### Arguments

| Name | Type | Description | Required |
|----------|------|-------------|----------|
| `input` | `[Segment; 2+]` | The line segments that should remain parallel. | Yes |
