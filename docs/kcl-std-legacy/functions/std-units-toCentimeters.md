Convert a number to centimeters from its current units.

```kcl
units::toCentimeters(@num: number(Length)): number(cm)
```



### Arguments

| Name | Type | Description | Required |
|----------|------|-------------|----------|
| `num` | `number(Length)` | A number. | Yes |

### Returns

`number(cm)` - A number.
