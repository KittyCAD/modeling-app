Constrain a line to be horizontal.

```kcl
solver::horizontal(@input: Segment)
```



### Arguments

| Name | Type | Description | Required |
|----------|------|-------------|----------|
| `input` | [`Segment`](/docs/kcl-std/types/std-types-Segment) | The line segment that should remain horizontal. | Yes |
