Convert a number to meters from its current units.

```kcl
units::toMeters(@num: number(Length)): number(m)
```



### Arguments

| Name | Type | Description | Required |
|----------|------|-------------|----------|
| `num` | `number(Length)` | A number. | Yes |

### Returns

`number(m)` - A number.
