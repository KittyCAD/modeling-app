Compute the minimum of the given arguments.

```kcl
min(@input: [number; 1+]): number
```



### Arguments

| Name | Type | Description | Required |
|----------|------|-------------|----------|
| `input` | `[number; 1+]` | An array of numbers to compute the minimum of. | Yes |

### Returns

`number` - A number.
