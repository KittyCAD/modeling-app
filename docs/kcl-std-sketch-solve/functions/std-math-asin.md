Compute the arcsine of a number.

```kcl
asin(@num: number(_)): number(rad)
```



### Arguments

| Name | Type | Description | Required |
|----------|------|-------------|----------|
| `num` | `number(_)` | A number. | Yes |

### Returns

`number(rad)` - A number.
