Converts a number to radians from its current units.

```kcl
units::toRadians(@num: number(Angle)): number(rad)
```



### Arguments

| Name | Type | Description | Required |
|----------|------|-------------|----------|
| `num` | `number(Angle)` | A number. | Yes |

### Returns

`number(rad)` - A number.
