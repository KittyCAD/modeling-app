Compute the maximum of the given arguments.

```kcl
max(@input: [number; 1+]): number
```



### Arguments

| Name | Type | Description | Required |
|----------|------|-------------|----------|
| `input` | `[number; 1+]` | An array of numbers to compute the maximum of. | Yes |

### Returns

`number` - A number.
