Compute the number to a power.

```kcl
pow(
  @input: number,
  exp: number(_),
): number
```



### Arguments

| Name | Type | Description | Required |
|----------|------|-------------|----------|
| `input` | `number` | The number to raise. | Yes |
| `exp` | `number(_)` | The power to raise to. | Yes |

### Returns

`number` - A number.
