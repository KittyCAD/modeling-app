Compute the four quadrant arctangent of Y and X.

```kcl
atan2(
  y: number(Length),
  x: number(Length),
): number(rad)
```



### Arguments

| Name | Type | Description | Required |
|----------|------|-------------|----------|
| `y` | `number(Length)` | A number. | Yes |
| `x` | `number(Length)` | A number. | Yes |

### Returns

`number(rad)` - A number.
