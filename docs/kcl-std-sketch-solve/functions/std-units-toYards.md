Converts a number to yards from its current units.

```kcl
units::toYards(@num: number(Length)): number(yd)
```



### Arguments

| Name | Type | Description | Required |
|----------|------|-------------|----------|
| `num` | `number(Length)` | A number. | Yes |

### Returns

`number(yd)` - A number.
