Compute the arctangent of a number.

```kcl
atan(@num: number(_)): number(rad)
```

Consider using `atan2()` instead for the true inverse of tangent.

### Arguments

| Name | Type | Description | Required |
|----------|------|-------------|----------|
| `num` | `number(_)` | A number. | Yes |

### Returns

`number(rad)` - A number.
