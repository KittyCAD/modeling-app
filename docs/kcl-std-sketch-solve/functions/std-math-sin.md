Compute the sine of a number.

```kcl
sin(@num: number(Angle)): number
```



### Arguments

| Name | Type | Description | Required |
|----------|------|-------------|----------|
| `num` | `number(Angle)` | A number. | Yes |

### Returns

`number` - A number.
