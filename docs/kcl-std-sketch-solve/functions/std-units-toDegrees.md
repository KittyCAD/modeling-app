Converts a number to degrees from its current units.

```kcl
units::toDegrees(@num: number(Angle)): number(deg)
```



### Arguments

| Name | Type | Description | Required |
|----------|------|-------------|----------|
| `num` | `number(Angle)` | A number. | Yes |

### Returns

`number(deg)` - A number.
