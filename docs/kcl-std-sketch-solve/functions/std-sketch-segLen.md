Compute the length of the provided line segment.

```kcl
segLen(@tag: TaggedEdge): number(Length)
```



### Arguments

| Name | Type | Description | Required |
|----------|------|-------------|----------|
| `tag` | `TaggedEdge` | The line segment being queried by its tag. | Yes |

### Returns

`number(Length)` - A number.
