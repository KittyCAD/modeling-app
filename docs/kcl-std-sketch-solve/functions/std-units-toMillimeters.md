Convert a number to millimeters from its current units.

```kcl
units::toMillimeters(@num: number(Length)): number(mm)
```



### Arguments

| Name | Type | Description | Required |
|----------|------|-------------|----------|
| `num` | `number(Length)` | A number. | Yes |

### Returns

`number(mm)` - A number.
