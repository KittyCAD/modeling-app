Compute the cosine of a number.

```kcl
cos(@num: number(Angle)): number
```



### Arguments

| Name | Type | Description | Required |
|----------|------|-------------|----------|
| `num` | `number(Angle)` | A number. | Yes |

### Returns

`number` - A number.
