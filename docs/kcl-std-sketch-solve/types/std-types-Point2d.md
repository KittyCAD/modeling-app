A point in two dimensional space.

```kcl
type Point2d = [number(Length); 2]
```

`Point2d` is an alias for a two-element array of numbers. To write a value
with type `Point2d`, use an array, e.g., `[0, 0]` or `[5.0, 3.14]`.
