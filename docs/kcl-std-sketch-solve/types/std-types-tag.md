**WARNING:** This type is deprecated.

Reference a previously created tag. Used much like a variable.

```kcl
type tag = TaggedEdge
```

Prefer to use `TaggedEdge` or `TaggedFace`. For more details on tags, see the docs for `TagDecl`.
