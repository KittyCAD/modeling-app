Given a hole profile, revolve it into a 3D hole solid geometry. This can be subtracted from a solid to form a hole in it.

```kcl
hole::revolveHole(
  @holeProfile,
  edge,
)
```



### Arguments

| Name | Type | Description | Required |
|----------|------|-------------|----------|
| `holeProfile` |  |  | Yes |
| `edge` |  |  | Yes |
