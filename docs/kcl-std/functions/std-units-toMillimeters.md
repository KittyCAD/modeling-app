Convert a number to millimeters from its current units.



```kcl
units::toMillimeters(@num: number(mm)): number(mm)
```


### Arguments

| Name | Type | Description | Required |
|----------|------|-------------|----------|
| `num` | `number(mm)` |  | Yes |

### Returns

`number(mm)`
