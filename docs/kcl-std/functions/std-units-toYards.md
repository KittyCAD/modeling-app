Converts a number to yards from its current units.



```kcl
units::toYards(@num: number(yd)): number(yd)
```

### Arguments

| Name | Type | Description | Required |
|----------|------|-------------|----------|
| `num` | `number(yd)` |  | Yes |

### Returns

`number(yd)`
