Data for revolution surfaces.


`object`

**Type:** `object`





## Properties

| Property | Type | Description | Required |
|----------|------|-------------|----------|
| `angle` | `number` (`double`)
 | Angle to revolve (in degrees). Default is 360. | No |
| `axis` | anyOf
 | Axis of revolution. | No |
| `tolerance` | `number` (`double`)
 | Tolerance for the revolve operation. | No |
