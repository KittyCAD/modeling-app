Computes the cosine of a number (in radians).



```js
cos(num: number) -> number
```

### Tags

* `math`

### Examples

```js
const anotherVar = cos(2 * pi())
```

### Arguments

* `num`: `number` (REQUIRED)

### Returns

`number`
