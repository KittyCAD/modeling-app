```js

```

| Name | Type | Description | Required |

| `hypotenuse` | [`number`](/docs/kcl/types/number) |  | Yes |
| `leg` | [`number`](/docs/kcl/types/number) |  | Yes |

### Returns

[`number`](/docs/kcl/types/number)


### Examples
