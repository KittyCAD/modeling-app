Convert a number to centimeters from its current units.



```js
toCentimeters(@num: number(cm)): number(cm)
```


### Arguments

| Name | Type | Description | Required |
|----------|------|-------------|----------|
| `num` | `number(cm)` |  | Yes |

### Returns

`number(cm)`
