Computes the number to a power.



```
pow(num: number, pow: number) -> number
```

### Examples

```kcl
const myVar = pow(4, 2)
```

### Arguments

* `num`: `number` (REQUIRED)
* `pow`: `number` (REQUIRED)

### Returns

`number`
