Converts a number from radians to degrees.



```js
toDegrees(num: number) -> number
```

### Tags

* `math`

### Examples

```js
const myVar = toDegrees(2 * pi())
```

### Arguments

* `num`: `number` (REQUIRED)

### Returns

`number`
