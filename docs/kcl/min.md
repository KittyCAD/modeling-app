Computes the minimum of the given arguments.



```js
min(args: [number]) -> number
```

### Tags

* `math`

### Examples

```js
const myVar = min(4, 5, 6)
```

### Arguments

* `args`: `[number]` (REQUIRED)

### Returns

`number`
