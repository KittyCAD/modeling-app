Import format specifier




**This schema accepts exactly one of the following:**

Autodesk Filmbox (FBX) format


**Type:** `object`




## Properties

| Property | Type | Description | Required |
|----------|------|-------------|----------|
| `type` |enum: `fbx`
|  | No |
