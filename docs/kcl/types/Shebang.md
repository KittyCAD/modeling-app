A shebang. This is a special type of comment that is at the top of the file. It looks like this: ```python,no_run #!/usr/bin/env python ```

**Type:** `object`





## Properties

| Property | Type | Description | Required |
|----------|------|-------------|----------|
| `content` |`string`|  | No |
| `start` |`integer`|  | No |
| `end` |`integer`|  | No |
