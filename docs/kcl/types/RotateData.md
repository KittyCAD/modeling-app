Data for rotating a solid with roll, pitch, and yaw.

**Type:** `object`





## Properties

| Property | Type | Description | Required |
|----------|------|-------------|----------|
| `roll` |`number` (**maximum:** 360.0) (**minimum:** -360.0)| The roll. | No |
| `pitch` |`number` (**maximum:** 360.0) (**minimum:** -360.0)| The pitch. | No |
| `yaw` |`number` (**maximum:** 360.0) (**minimum:** -360.0)| The yaw. | No |
| `global` |`boolean`| If true, the transform is applied in global space. If false, the transform is applied in local sketch axis. | No |
