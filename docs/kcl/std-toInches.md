Convert a number to inches from its current units.



```js
toInches(@num: number(in)): number(in)
```


### Arguments

| Name | Type | Description | Required |
|----------|------|-------------|----------|
| `num` | `number(in)` |  | Yes |

### Returns

`number(in)`
