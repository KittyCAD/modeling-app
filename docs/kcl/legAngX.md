Compute the angle of the given leg for x.



```js
legAngX(hypotenuse: number, leg: number) -> number
```

### Tags

* `utilities`


### Arguments

| Name | Type | Description | Required |
|----------|------|-------------|----------|
| `hypotenuse` | `number` |  | Yes |
| `leg` | `number` |  | Yes |

### Returns

`number` 


### Examples

```js

```
